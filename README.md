# Billiard Average Tracker

A simple Tkinter-based app for tracking billiard averages.

## Features
- Supports Straight Rail, 1 Cushion, and 3 Cushion
- Tracks last 30 matches per game type
- Calculates weighted averages
- Undo last match
- Highlights best and worst performances
